This plugin create a widget for countdown.

== Description ==

A Simple countdown widget based on [jQuery Countdown](http://keith-wood.name/countdown.html). Will have customizeable css with next versions. Multilanguage.

Plugin is multilingual. If you translate [please contact me](mailto:info@wpadami.com) and send files.

= Languages Included (Plugin) =
* Turkish (tr_TR)	(Default)
* English (en_US)	(Default)
* Serbian (sr_RS)	[Borisa Djuraskovic](http://www.webhostinghub.com/)
* French (fr_FR)	(Default)

== Installation ==

1. Upload `cwidget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Widgets` and set your options.
